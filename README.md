### 使用方法
1. 克隆项目到本地
2. 安装依赖
   ```bash
   pnpm i
   ```

3. 根据你使用的框架修改主进程文件`src/main/index.ts`中导入插件框架的那行
4. 运行项目
   ```bash
   pnpm dev
   ```
